Deskripsi Singkat

Saya Dian Siswandi, Lulus di salah satu Perguruan Tinggi Swasta yaitu: Sekolah Tinggi Manajemen dan Ilmu Komputer Musi Rawas atau yang biasa disebut STMIK MURA Lubuklinggau dan sekarang berganti nama Universitas Bina Insan atau disebut UNIBI. Menjadi Operator Entri Data disebuah Instansi Pemerintah di Kabupaten Musi Rawas Utara; 2. Menjadi Operator Server Quick Count Pemilihan Bupati dan Presiden; Dan ketika lulus kuliah saya pernah membuat website Jual Beli (web dinamis/web sederhana) dibeberapa tempat dan saya jual ke teman dan relasi lainnya. Dan tahun setelah lulus itu saya bekerja juga sebagai Karyawan Honorer dan menduduki jabatan sebagai Administrator Database 1 tahun dan kemudian menjadi Operator Keuangan sampai dengan sekarang. Saya sangat hobi dengan membaca referensi pelajaran di media online, membuat desain gambar, dan membuat website sejak saya duduk di bangku Sekolah Menengah Atas. Mungkin ini saja deskripsi singkat saya, terima kasih.
